# Pucca Coffee

A playful Pucca-inspired coffee landing page featuring a themed hero section, hot and iced signature drinks, and a responsive gallery.

## Live preview

- GitHub Pages: https://zaichi234509.github.io/pucca/

## What's included

- `index.html` — single-page static website
- `assets/` — coffee and Pucca-themed visual assets

## Run locally

Open `index.html` in your browser, or serve the folder locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
